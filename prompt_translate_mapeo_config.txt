Abaixo está um exemplo de um arquivo CSV. Neles estão idiomas. algumas linhas são categorias. As outras linhas são as traduções. Escreva um script node Que pegue todos os arquivos JSON de uma pasta, que é o primeiro parametro. Selecione o campo "name" de cada arquivo json, veja qual dos valores da primeira coluna do arquivo CSV são mais similares a esse valor. Crie novas versões desses arquivos com as traduções para as outras quatro línguas. Todos outros valores deveram permanecer o mesmo. O Segundo parametro é qual o path de ouput dos arquivos json. Deve ser um script bin que aceita parametros.

Também crie instruções de uso em inglês, como arquivo README.md do Github.
